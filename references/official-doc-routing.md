# SylixOS 官方文档路由

当任务涉及 SylixOS 运行时能力、Shell、应用开发、驱动开发、容器开发或相关调试时，优先按主题进入官方文档对应子站点，而不是在整站里盲搜。

## 入口

- Shell 文档：https://docs.acoinfo.com/sylixos/shell/
- 应用开发文档：https://docs.acoinfo.com/sylixos/app/
- 驱动开发文档：https://docs.acoinfo.com/sylixos/drv/
- ECS 容器文档：https://docs.acoinfo.com/sylixos/ecs/

## 何时选哪个入口

### `shell`

适用场景：

- Shell 命令语法和边界
- 目录、文件、用户、网络、进程、线程、内存、时间相关命令
- `/etc`、`/proc`、文件系统、网络配置、挂载、路由
- Shell 扩展和自定义命令

从该入口可继续找：

- `登录命令行`
- `命令行相关命令`
- `添加自定义 Shell 命令`
- 具体命令页面，例如 `ls`、`cat`、`ps`、`ifconfig`、`route`

### `app`

适用场景：

- 用户态应用开发与调试
- 进程、线程、IPC、信号、时间、内存、网络、文件系统
- 动态装载、标准 IO、日志、平台迁移
- upload 后验证应用行为，或需要构造最小 app 复现

从该入口可继续找：

- `编写第一个程序`
- `线程管理`
- `进程管理`
- `网络通信`
- `文件系统`
- `动态装载`

### `drv`

适用场景：

- 内核态调试
- `ko`、驱动、BSP、设备树
- 中断、时钟、Cache/MMU、字符设备、GPIO、DMA、网络设备
- upload 后验证模块装载或驱动行为，或需要构造最小 `ko` 复现

从该入口可继续找：

- `构建第一个驱动程序`
- `PROC 文件系统`
- `中断系统与时钟`
- `字符设备驱动`
- `网络设备驱动`
- `板级支持包`
- `设备树`

### `ecs`

适用场景：

- SylixOS ECS 容器命令与容器运行时
- 容器配置、环境变量、环境隔离
- 镜像、仓库、Bundle、容器节点
- 容器日志、容器状态、容器内执行与调试

从该入口可继续找：

- `快速入门`
- `容器命令`
- `容器配置`
- `环境变量`
- `环境隔离`
- `镜像管理`
- `查看调试日志`

## 选路规则

- Shell 命令不会写时，先去 `shell`
- 用户态二进制、共享库、动态装载问题，先去 `app`
- 驱动、模块、BSP、设备树、中断类问题，先去 `drv`
- 容器命令、镜像、节点、日志、容器内执行，先去 `ecs`
- 一个问题跨多个域时，先按主问题选入口；例如“upload 后 telnet 登录查看进程”先 `shell`，“upload 后运行 app 异常”先 `app`，“upload 后装载 ko 失败”先 `drv`
